# Safe Squat Tutor

## Prerequisites
- Python uv package manager
- Adequate VRAM to run the MediaPipe inference engine
- WebCam

## Installation
1. Use `git clone` to clone this git repository locally.
2. Retrieve a pose landmarker model from https://ai.google.dev/edge/mediapipe/solutions/vision/pose_landmarker/index#models. Place the file in the root of the cloned project.
3. Invoke `uv run app.py`
